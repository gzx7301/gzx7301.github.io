0.Requirements

未修改 已解压的游戏资源文件
Game Tools
已经脱壳了的amdaemon.exe
脱壳了的游戏主程序
ODD.SYS
osrloader(使用Google寻找即可)

或使用Game Tools + ODD.sys + odd-loader.bat

1.获取需要的工具

  • 游戏需要启动工具才可正常启动

工具在这个地方下载
→|Game Tools|←
→|ODD.ZIP(ODD.sys + odd-loader.bat)|←
下载内容 

Game Tools.zip 
odd.zip

2.操作

1.准备游戏,载入ODD.sys

解压odd.zip至任意非中文路径文件夹 以管理员身份运行bat 提示按下任意按键继续时为加载成功 若按下了任意按键并继续 则会自动卸载odd.sys

在刚刚下载的 Game Tools.zip 中 寻找到与你所需要启动的游戏对应工具的并 覆盖
Windows下 此操作完成后应出现第1个命令行窗口

将 amdaemon.exe 更换为脱壳后版本

2. 启动 获取资源

在游戏目录里打开CMD(SHELL) 运行 

inject -d -k cmhook.dll amdaemon.exe -c config_common.json config_server.json config_client.json config_hook.json

Windows下 此操作完成后应出现第2个命令行窗口

运行游戏主程序 游戏应出现加载中的表现 但此时不会加载成功
